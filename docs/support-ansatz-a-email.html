Setup-Schritte

  • 1 all-inkl KAS → E-Mail → Neues Postfach: support@korrekturpilot.de anlegen
  • 2 Weiterleitung einrichten: Kopie an melissaconrads@gmail.com + christopher@...
  • 3 Autoresponder aktivieren (KAS → E-Mail → Autoresponder) mit obigem Text
  • 4 Im Code alle "kontakt@korrekturpilot.de" durch "support@korrekturpilot.de" ersetzen
  • 5 Supabase Auth Absender-Adresse auf support@ aendern (Dashboard → Auth → Email Templates)

Bewertung

Vorteile

  • Keine Zusatzkosten
  • In 15 Min eingerichtet
  • Keine neuen Tools zu lernen
  • DSGVO-konform (all-inkl = DE)
  • Funktioniert sofort

Nachteile

  • Kein Ticket-Tracking
  • Keine Statusverfolgung (offen/geschlossen)
  • Kein gemeinsamer Posteingang
  • Skaliert nicht ueber 50+ Anfragen/Woche
  • Screenshots per Mail umstaendlich
